What is Cloud Computing?

Cloud computing is the on-demand availability of computer system resources—especially data storage and computing power—without direct active management by the user. It allows organizations to rent access to everything from applications to storage from a cloud service provider.

There are three main types of cloud services:

  • SaaS – Software as a Service
  • PaaS – Platform as a Service
  • IaaS – Infrastructure as a Service
